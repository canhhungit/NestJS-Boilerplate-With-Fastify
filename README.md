## Description

This is a NestJS boilerplate.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

# compodoc
$ npx @compodoc/compodoc -p tsconfig.json -s
```

## Links

```bash
# Localhost backend :
$ http://localhost/api

# Swagger :
$ http://localhost/docs/

# Compodoc
$ http://127.0.0.1:8080/
```
